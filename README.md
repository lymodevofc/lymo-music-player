# 🍋 Lymo Music Player

Lymo Music Player is a lightweight, Electron-based desktop music player with a playful, retro-inspired interface — featuring a signature spinning lemon animation and a tactile rotary volume knob.

## ✨ Features

- 📁 **Folder-based music scanning** — point it at a folder and it builds your library automatically
- 🎶 **Playlist creation and management** — organize your tracks into custom playlists
- 🍋 **Spinning lemon animation** — a fun, animated visual centerpiece while music plays
- 🖼️ **Album art support** — displays embedded cover art for your tracks
- 🎛️ **Rotary volume knob** — a satisfying, skeuomorphic volume control
- 🔍 **Song search** — quickly find tracks across your library
- 🔀 **Shuffle and repeat modes** — listen your way

## 🚀 Installation

Make sure you have [Node.js](https://nodejs.org/) installed, then run:

```bash
npm install
npm start
```

This will install all dependencies and launch Lymo Music Player.

## 📦 Build

To package the app into a distributable installer:

```bash
npm run dist
```

The built installer will be output to the `dist/` folder.
